# AI Budget Monitoring System

An enterprise-level **AI-powered Budget Monitoring System** developed during the **Unified Mentor Internship Program**.

The application enables organizations to efficiently manage departmental budgets, monitor expenditures, generate intelligent financial alerts, visualize budget utilization using interactive dashboards, and produce detailed reports.

Built using **Angular 22**, **Node.js**, **Express.js**, and **MongoDB Atlas**, the system demonstrates a modern full-stack architecture with secure authentication and responsive design.

# Installation

## Clone Repository

```bash
git clone https://github.com/DurgeshBodke/AI-Budget-Monitoring-System.git
```

---

## Backend

```bash
cd server

npm install

npm start
```

---

## Frontend

```bash
cd client

npm install

ng serve
```

Open

```
http://localhost:4200
```

---

# Environment Variables

Create a `.env` file inside the **server** folder.

```
PORT=5000

MONGO_URI=YOUR_MONGODB_ATLAS_CONNECTION_STRING

JWT_SECRET=YOUR_SECRET_KEY
```
