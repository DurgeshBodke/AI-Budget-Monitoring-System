## Overview

AI Budget Monitoring System is a full-stack web application developed to help organizations monitor budget allocation, expenditure tracking, financial utilization and AI-driven budget analysis.

The system provides real-time monitoring of departmental budgets, tracks expenses, generates intelligent alerts, and visualizes financial data using interactive charts.

---

## Features

- User Authentication (JWT)
- Department Management
- Budget Allocation
- Expense Tracking
- AI Budget Insights
- Budget Utilization Analysis
- Intelligent Alerts
- Reports Dashboard
- Real-time Statistics
- File Upload Support
- Interactive Charts
- Responsive UI

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Budget-Monitoring-System.git
```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
ng serve
```

---

## Environment Variables

Create a `.env` file inside the server folder.

```
PORT=5000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY
```
